Опции
Если эта функция включена, основная панель кнопок будет расположена в верхней части интерфейса, а не в нижней.
Если этот параметр включен, все значения файла cookie отображаются в интерфейсе.
Если этот параметр отключен, в интерфейсе отображаются только имя и значение файла cookie.
Когда включено, в интерфейсе Cookie Checker будут отображаться переходы и другие различные анимации. Это только для внешнего вида.
Выбирает режим работы кнопки экспорта. Если выбрано значение "Вручную", при нажатии кнопки экспорта отобразится меню с запросом на выбор формата.
Выбирает один дополнительный элемент для отображения рядом с названием файла cookie в интерфейсе. Это поможет вам быстро определить, какой файл cookie вы ищете.
При выборе "Автоматически" , Cookie Checker будет автоматически соответствовать тематике вашего браузера.
Все Cookies
Будьте осторожны! Действия в этом разделе будут применяться ко всем сайтам.
Помните о правилах безопасности ваших персональных данных.
Экспорт всех Cookies
Header string недоступен для экспорта из браузера, поскольку он не содержит информации о том, с какого сайта он взят.
Удалить все Cookies
Это приведет к удалению всех файлов cookie на всех сайтах в этом браузере. Это действие необратимо. Будьте очень осторожны.
